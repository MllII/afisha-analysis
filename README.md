# \##  Описание проекта

# 

# В этом проекте я проанализировала данные сервиса по продаже билетов на мероприятия (концерты, театры, спортивные события и др.).

# \*\*Основная цель\*\* — выявить ключевые факторы, которые заставляют пользователей возвращаться на платформу за повторной покупкой, и дать рекомендации по улучшению удержания клиентов.

# \##  Основные выводы исследования

# 

# \### 1. Общая статистика

# \- \*\*21 854 пользователя\*\* совершили покупки за анализируемый период.

# \- \*\*61.7%\*\* пользователей возвращаются и совершают \*\*2+ заказа\*\*.

# \- \*\*Критический период удержания:\*\* первые \*\*14 дней\*\* после первого заказа.

# 

# \### 2. Влияние количества билетов

# \- \*\*"Золотая середина":\*\* пользователи, покупающие \*\*2–3 билета\*\*, возвращаются чаще всего (\*\*74.2%\*\*).

# \- \*\*Риск:\*\* пользователи, покупающие \*\*5+ билетов\*\*, почти не возвращаются (всего \*\*18.8%\*\*) — это, вероятно, разовые групповые покупки.

# 

# \### 3. Временные интервалы

# \- \*\*46.9%\*\* возвратов происходят в течение \*\*7 дней\*\*.

# \- \*\*65.2%\*\* возвратов происходят в течение \*\*14 дней\*\*.

# \- Если перерыв между покупками превышает 30 дней, пользователь попадает в зону риска оттока.

# 

# \### 4. Тип мероприятия и возврат

# \- Лучшие точки входа: \*\*выставки\*\* (64.4%) и \*\*театр\*\* (63.8%).

# \- Худшие точки входа: \*\*ёлки\*\* (55.8%) и \*\*спорт\*\* (56.2%).

# 

# \### 5. Корреляционный анализ

# \- \*\*Средний интервал между заказами (`avg\_days\_between`)\*\* — самый сильный предиктор количества заказов.

# \- Категориальные признаки (регион, оператор) практически не влияют на частоту покупок.
